🎓 Placement Management System 🏢
📌 Project Overview
The Placement Management System is a web-based application designed to streamline the placement process for students and recruiters. It automates student registrations, job postings, and interview schedules, making the process smooth and hassle-free!

✨ Features
✅ Admin Panel - Manage students, recruiters, and job postings.
✅ Student Portal - Register, update profiles, and apply for jobs.
✅ Recruiter Portal - Post job openings and view student applications.
✅ Secure Authentication - Login and registration for students & recruiters.
✅ Database Management - Store student details, job listings, and recruiter information.

📁 Project Folder Structure

📂 .github/                # GitHub-related configuration files

📄 forgot.php             # Password recovery system

📄 login.php              # User login page

🛠 Technologies Used
💻 Frontend: HTML, CSS
⚙️ Backend: PHP
🗄 Database: MySQL
🐙 Version Control: Git & GitHub


🚀 Installation Guide
1️⃣ Clone the repository
git clone https://github.com/taheer007/Placement-Management-System.git

2️⃣ Move into the project directory
cd Placement-Management-System

3️⃣ Set up a local server (XAMPP, WAMP, or any preferred one).

4️⃣ Import the database (sql folder) into MySQL.

5️⃣ Start the Apache & MySQL servers.

6️⃣ Open the project in a browser
http://localhost/Placement-Management-System
