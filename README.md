Project Overview:

The Placement Management System is a web-based application designed to streamline the placement process for students and recruiters. It helps manage student registrations, job postings, and interview schedules efficiently.

Features:

Admin Panel: Manage students, recruiters, and job postings.

Student Portal: Register, update profiles, and apply for jobs.

Recruiter Portal: Post job openings and view student applications.

Authentication System: Secure login and registration for students and recruiters.

Database Management: Store student details, job listings, and recruiter information.

Folder Structure:

.github/                     # GitHub-related configuration files
admin/                       # Admin dashboard and management files
css/                         # Stylesheets for UI design
images/                      # Image assets used in the project
includes/                    # Common files like header, footer, and database connection

Screenshots/                 # Screenshots of the project (if available)

.gitattributes               # Git LFS tracking configuration

forgot.php                   # Password recovery system
index.php                    # Homepage
login.php                    # User login page
register.php                 # User registration page

Technologies Used:

Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Version Control: Git & GitHub

Installation Guide:

Clone the repository:

git clone https://github.com/taheer007/Placement-Management-System.git

Move to the project directory:

cd Placement-Management-System

Setup a local server (XAMPP, WAMP, etc.).

Import the database (sql folder) into MySQL.

Start the Apache and MySQL servers.

Open the project in a web browser:

http://localhost/Placement-Management-System
